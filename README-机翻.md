# 岩原爬行者

## iwara_crawler_v2.py

自动下载特定用户的所有视频。

***警告**：此脚本中使用了帐户信息。*

*Iwara.com 可以轻松找出哪个帐户正在抓取视频。*

*可能会导致您的帐户被禁止/封锁。没有任何保证。*

#### **你被警告了**

### 如何使用

1. 克隆或下载此存储库
2. pip3 install -r 要求
3. 在py脚本中设置用户和代理信息
4. python3 iwara_crawler_v2.py
5. 在弹出的浏览器中手动登录（仅限第一次）
6. 注意剩余磁盘空间并祝你玩得开心:)

但只产生url，必须手动一一下载。
